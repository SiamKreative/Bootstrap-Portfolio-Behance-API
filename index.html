Bootstrap Portfolio
with Behance API & jQuery

If you're using Behance to showcase your projects and you would like to embed your portfolio on your site, look no further.

This quick example show you how to retrieve your projects from Behance using their API, store the data in the browser, and display it using Bootstrap 3 markup. To render the template we use jQuery, but you could also use a template engine such as pure.js or handlebars. Find the most suitable template engine here.
